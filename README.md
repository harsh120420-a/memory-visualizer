# 🧠 Dynamic Memory Management Visualizer

A modern, interactive web application built to simulate and visualize Operating System memory management techniques, including **Virtual Memory (Paging)** and **Segmentation**.

## 🚀 Features

### 📟 Virtual Memory (Paging)
- **Algorithm Support:** FIFO, LRU, and Optimal Page Replacement.
- **Real-time Visualization:** Watch page hits and faults occur as the reference string is processed.
- **Interactive Presets:** Includes a "Load Belady's Anomaly" button to demonstrate counter-intuitive OS behavior.

### 🍱 Segmentation
- **Allocation Strategies:** First Fit, Best Fit, and Worst Fit.
- **Fragmentation Tracking:** Real-time detection of **External Fragmentation**.
- **Compaction:** Simulate memory defragmentation to reclaim scattered free space.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
